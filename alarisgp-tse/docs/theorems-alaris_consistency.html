Module: alaris_consistency

Consistency theorems.
Consistency Templates:
\forall a in A_c, s in S, m in MS guard(s, m) => pre_filter(s, m) R post_filter(a(s), m)
where: A_c is a set of actions \subseteq S -> S

State consistency \forall s in S: guard(s) => invariant(s)
